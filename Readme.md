# React Snake Game

A simple snake game made with React using the canvas element.

## How to play

- Use the arrow keys or WASD to move the snake around the screen

## How to run

- Clone the repo
- Run `npm install`
- Run `npm start`
- Open `http://localhost:5173` in your browser
